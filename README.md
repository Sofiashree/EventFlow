# EventFlow – Event Management System

A full-stack Event Management System developed using Spring Boot, React, MySQL, HTML, CSS, and JavaScript. The application enables users to browse events, book tickets, and manage bookings through a responsive web interface.

## Features

- User Registration & Login
- Browse Available Events
- Event Booking
- View My Bookings
- Admin Event Management
- RESTful APIs
- Responsive User Interface
